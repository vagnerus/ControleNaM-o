## Core Features:

- Transaction Recording: Manually add expenses and income, specifying amount, date, and description.
- Categorization: Categorize expenses and income with customizable categories and subcategories.
- Recurring Transactions: Automatically register fixed expenses (like rent) and fixed income (salary) every month.
- Credit Card Management: Visualize the current invoice amount, available limit and closing date.
- Installment Launch: When launching an installment purchase, the app automatically projects the installments in future months.
- Monthly Budgets: Define a spending ceiling for each category and track progress. Display of potential future problems suggested by an AI tool.
- Financial Goals: Set a goal, total amount and monthly saving amount. The app monitors progress.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to convey trust and financial responsibility.
- Background color: Light blue (#E0F7FA), a desaturated variant of the primary color to establish a light color scheme.
- Accent color: Violet (#7C4DFF) for highlighting important actions.
- Body and headline font: 'Inter' for a modern, machined, objective, neutral feel.
- Use financial-themed icons for categories and actions.
- Clean, minimalist layout with clear separation of sections for easy navigation.
- Subtle transitions and animations for a smooth user experience.
